=== Формат хранения

* ini
* yaml
* динамический

=== INI формат

```ini
[global_server]
example.com hostname=example.com

[local_server]
mail hostname=mail.corp

[servers:children]
global_server
local_server

```

=== YAML формат

```yaml
global_server:
  hosts:
    example.com:
      hostname: example.com

local_server:
  hosts:
    mail:
      hostname: mail.corp
servers:
  children:
    global_server:
    local_server:
```
